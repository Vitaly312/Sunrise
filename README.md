# Sunrise
***Sunrise*** - это веб-сайт для управления личными задачами(ToDo - сервис).
Сайт позволяет пользователям создать список задач и отмечать 
выполненные задачи, что позволяет удобно организовать свои задачи.

Сайт написан на Python, с использованием фреймворка ```Flask```

## Установка и запуск
1. Скачайте репозиторий
2. Выполните: ```docker-compose up -d```

После этого, сайт будет доступен по адресу http://localhost:80/


Изменить порт можно в файле ```docker-compose.yml```

### Переменные окружения

Вы можете задать значения переменных в файле ```.env.example```
Переменные:
- ```SECRET_KEY``` - значение ключа FLASK_SECRET
- ```DB_PROD``` - URI для подключения к базе данных(SQLAlchemy)
